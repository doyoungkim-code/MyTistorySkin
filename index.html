## 브라우저 창
드래그로 이동 / 모서리로 크기 조절
닫기(빨강) · 최소화(노랑) · 최대화(초록)
뒤로 · 앞으로 · 새로고침 지원
## Blogram (인스타그램 스타일 사진 피드)
사진 카테고리 포스트를 Instagram UI로 감상
프로필 헤더 — 아바타 · 게시물 수 · 팔로워 · 팔로잉
3열 그리드 썸네일, 사진 클릭 시 라이트박스
라이트박스 — 좋아요 · 댓글 · 공유 링크 복사
좋아요: 로그인 / 비로그인 모두 가능 (쿠키 중복 방지)
댓글: 게스트는 이름 + 비밀번호 입력 후 작성
← → 키 또는 버튼으로 이전 / 다음 사진 이동
## Guestbook (방명록 채팅)
카카오톡 스타일 채팅 인터페이스
방문자 메시지 → 오른쪽 노란 말풍선
관리자 답글 → 왼쪽 흰 말풍선
이름 + 비밀번호 입력 후 메시지 전송
Ctrl+Enter 로 빠른 전송
## File Explorer (블로그 카테고리 탐색기)
블로그 카테고리 구조를 파일 탐색기 UI로 탐색
사이드바 트리 + 메인 영역으로 구성
Grid 뷰 / List 뷰 전환 버튼 지원
뒤로 · 앞으로 내비게이션 및 주소바 표시
폴더 클릭 → 하위 카테고리로 이동
파일(최하위 카테고리) 클릭 → 브라우저 창으로 열기
우클릭 → Open / Properties 메뉴 사용 가능
## Music Player (YouTube 기반 뮤직 플레이어)
YouTube 영상을 배경 음악으로 재생하는 플레이어
앨범 아트 · 곡 제목 · 채널명 표시
재생 / 일시정지 · 이전곡 / 다음곡 버튼
시크바로 원하는 구간으로 이동
볼륨 슬라이더로 음량 조절
플레이리스트에서 곡을 클릭하면 즉시 재생
