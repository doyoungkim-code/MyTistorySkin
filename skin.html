## 브라우저 창
드래그로 이동 / 모서리로 크기 조절
닫기(빨강) · 최소화(노랑) · 최대화(초록)
뒤로 · 앞으로 · 새로고침 지원
## Guestbook (방명록 채팅)
카카오톡 스타일 채팅 인터페이스
방문자 메시지 → 오른쪽 노란 말풍선
관리자 답글 → 왼쪽 흰 말풍선
이름 + 비밀번호 입력 후 메시지 전송
Ctrl+Enter 로 빠른 전송
## File Explorer (블로그 카테고리 탐색기)
블로그 카테고리 구조를 파일 탐색기 UI로 탐색
사이드바 트리 + 메인 영역으로 구성
Grid 뷰 / List 뷰 전환 버튼 지원
뒤로 · 앞으로 내비게이션 및 주소바 표시
폴더 클릭 → 하위 카테고리로 이동
파일(최하위 카테고리) 클릭 → 브라우저 창으로 열기
우클릭 → Open / Properties 메뉴 사용 가능
## Music Player (YouTube 기반 뮤직 플레이어)

♪ 버튼 클릭 시 플레이어 창 열기
재생 / 일시정지 · 볼륨 조절
앨범 아트 · 곡 제목 · 채널명 표시
플레이리스트에서 곡을 클릭하면 즉시 재생
